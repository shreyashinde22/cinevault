About StreamNet

StreamNet is a premier static UI prototype demonstrating a modern, responsive streaming platform interface. Built entirely with HTML5 and Bootstrap 5 without any custom JavaScript.

This project focuses on minimal CSS and clean structure, achieving a dark-themed, content-focused experience reminiscent of top streaming networks. No real backend or streaming capabilities are implemented.
